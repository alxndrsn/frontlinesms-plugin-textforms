TextForms

The TextForms allows complex data gathering and simple type validation via SMS through a familiar question and answer format.

Activating the TextForms plugin

When FrontlineSMS is installed for the first time, the TextForms module is initially hidden from view. To activate the TextForms module, go to the View menu, then select Tabs, and make sure "TextForms" is ticked. A new TextForms tab should then appear with your other menus along the top of the screen. To hide the TextForms module, go back into the same menu and remove the tick/check from the "TextForms" entry.

Manage Contacts

The Manage Contacts task allows you to add a new contact, edit an existing contact or delete a contact. Selecting a contact from the list will also display their details in the right pane.

Clicking the View Answers button will open the Browse Answers task for the currently selected Contact showing all the answers they have submitted.

Note, you can filter the contact list by entering a search term into the Search contacts... textbox.

Add / Edit Contact

Clicking the Add Contact button will present the Contact dialog allowing you to enter their Name, Phone, Organization and Email. To edit an existing contact, double-click a contact from the list or select a contact then click the Edit Contact button.
  • Name - the contact's name
  • Phone - the contact's phone
  • Organization - the contact's organization
  • Email - the contact's email

Manage Questions

The Manage Questions task allows you to add a new question, edit an existing question or delete a question. Selecting a question from the list will also display that question's details in the right pane.

Clicking the View Answers button will open the Browse Answers task for the currently selected Question showing all the answers that have been submitted for that Question.

Note, you can filter the question list by entering a search term into the Search questions... textbox.

Add / Edit Question

Clicking the Add Question button will present the Question dialog allowing you to enter the question Name, Keyword, Schema, Choices and Phrase. To edit an existing question, double-click a question from the list or select a question then click the Edit Question button.
  • Name - the name of the question
  • Keyword - the keyword used to distinguish this question
  • Schema - the schema name, used when optionally uploading the received answers to an external server
  • Type - the question type: Plain Text, Integer, Boolean, Date, Checklist, Multiple Choice
  • Choices - the options for Checklist and Multiple Choice type
  • Phrase - the question phrase used sent to the user asking what they are answering

Manage TextForms

The Manage TextForms task allows you to add a new textform, edit an existing textform or delete a textform. Selecting a textform from the list will also display that textform's details in the right pane.

Note, you can filter the textform list by entering a search term into the Search textforms... textbox.

Add / Edit TextForm

Clicking the Add TextForm button will present the TextForm dialog allowing you to enter the textform Name, Keyword and Questions. To edit an existing textform, double-click a textform from the list or select a textform then click the Edit TextForm button.
  • Name - the textform name
  • Keyword - the keyword used to distinguish this textform
  • Question - add or delete questions associated with this TextForm

Browse Answers

The Browse Answers task allows you to view and search through all the received responses.

You can filter contacts by entering a term into the Search contacts... textbox.

Manage Options

The Manage Options has three main sections:

Register / Help Keywords - the accepted keywords to trigger registering a mobile device or requesting help information
  • Register - add or remove accepted keywords to register a mobile device
  • Help - add or remove accepted keywords to request help information
Boolean Values - the accepted keywords for boolean values
  • True - add or remove accepted true values
  • False - add or remove accepted false values
Upload Options - the plugin can optionally upload received data to an external server
  • URL - the URL for external server you wish to upload to
  • Handler - the data type to be uploaded: XML, JSON, CSV or the Google specific XMl format for ResourceFinder
